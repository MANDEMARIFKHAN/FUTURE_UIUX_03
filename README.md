# Event Plaza CRM Dashboard UI

A modern CRM (Customer Relationship Management) Dashboard designed for event management agencies to efficiently manage leads, clients, follow-ups, tasks, and event workflows in one centralized platform.

## 📌 Project Overview

Event Plaza CRM is designed for small and medium-sized event management agencies handling multiple clients and events simultaneously. The dashboard streamlines client communication, lead tracking, task management, and event planning activities.

This project was created as part of **Future Interns UI/UX Design Task 3 (2026)**.

---

## 🎯 Problem Statement

Event management agencies often manage:

* Client inquiries
* Event bookings
* Follow-ups
* Team tasks
* Vendor coordination

Using spreadsheets and messaging apps can lead to missed follow-ups, scattered information, and inefficient workflows.

Event Plaza CRM solves these challenges through a centralized dashboard experience.

## 💡 Design Rationale

The CRM follows a workflow-based approach:

Lead → Contacted → Proposal → Client → Event Planning → Event Execution → Feedback

The interface prioritizes:

* Fast navigation
* Clear information hierarchy
* Scalable data management
* Productivity-focused workflows
